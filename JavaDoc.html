Calculator

Kelas Calculator menyediakan metode dasar untuk operasi matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian.

Method Summary

  • add(int a, int b) - Menambahkan dua bilangan dan mengembalikan hasilnya.
  • subtract(int a, int b) - Mengurangi satu bilangan dari bilangan lainnya dan mengembalikan hasilnya.
  • multiply(int a, int b) - Mengalikan dua bilangan dan mengembalikan hasilnya.
  • divide(int a, int b) - Membagi satu bilangan dengan bilangan lainnya dan mengembalikan hasilnya. Melempar ArithmeticException jika penyebut adalah 0.

Method Details

add

Parameters: int a - bilangan pertama, int b - bilangan kedua

Returns: hasil dari penjumlahan a dan b

subtract

Parameters: int a - bilangan pertama, int b - bilangan kedua

Returns: hasil dari pengurangan b dari a

multiply

Parameters: int a - bilangan pertama, int b - bilangan kedua

Returns: hasil dari perkalian a dan b

divide

Parameters: int a - bilangan pertama (pembilang), int b - bilangan kedua (penyebut)

Returns: hasil dari pembagian a dengan b

Throws: ArithmeticException jika b adalah 0
